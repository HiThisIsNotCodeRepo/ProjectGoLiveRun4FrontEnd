# Git

## Common Script
`git pull <remote addr>`
- To pull repo from remote

`git merge dev`
- To merge with branch

`git reset HEAD .`
`git checkout .`
- To restore files accidently deleted

`git reset HEAD .`
`git clean -df`
- To delete files accidently added

`git reset HEAD .`
`git checkout .`
- To bring the files to initial condition
